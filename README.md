# AI Stock Market Analyzer & Predictor

This project is an AI-powered stock market analysis and prediction web application built using Python, Streamlit, and LSTM deep learning.

The system analyzes historical stock data, visualizes trends, and predicts future stock prices using a Long Short-Term Memory (LSTM) neural network.

---

## Features

- Real-time stock data from Yahoo Finance
- Stock price visualization
- Moving average analysis
- LSTM-based stock price prediction
- Prediction vs actual comparison
- Model accuracy evaluation (RMSE)

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- yFinance
- TensorFlow / Keras
- LSTM Neural Networks
- Scikit-learn
